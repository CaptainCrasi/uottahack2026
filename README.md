# MarketSnipe (UottaHack 2026)

**MarketSnipe** is a web application designed to help small businesses find their exact target audience using **Yellowcake** Reddit scanning technology. It features a guest discovery mode and a data-driven brainstorming tool to identify market gaps.

## Features
- **Audience Discovery**: Scan Reddit to find users matching your project's criteria.
- **Market Gap Discovery**: "Find the Gap" tool (WOW Feature) identifying unmet needs.
- **Green Professional UI**: A clean, modern interface built with React Vite.
